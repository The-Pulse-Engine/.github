## Who we are

PulseEngine is a developer community building software that belongs to the people who use it.

No subscriptions. No cloud lock-in. No data harvesting. No paywalls. No compromises.

We believe the best software runs locally, respects your privacy by default, and costs nothing. That is not idealism. That is a design decision. And we make it on every project we ship.
